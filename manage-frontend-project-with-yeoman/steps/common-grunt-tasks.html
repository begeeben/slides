Common Grunt Tasks

  • contrib-connect: Start a connect web server.
  • contrib-watch: Run predefined tasks whenever watched file patterns are added, changed or deleted.
  • contrib-jshint: Validate files with JSHint.
  • contrib-copy: Copy files and folders.
  • usemin: Replace references to non-optimized scripts or stylesheets into a set of HTML files (or any templates/views).
  • rev: Static file asset revisioning through content hashing.
